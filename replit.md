# Discord Bot Management Dashboard

## Project Overview
A comprehensive Discord bot management system with web dashboard, database persistence, and keep-alive functionality. Features include moderation tools, music bot configuration, anti-nuke protection, and server management.

## Project Architecture
- **Frontend**: React with TypeScript, TailwindCSS, and shadcn/ui components
- **Backend**: Express.js with WebSocket support for real-time updates
- **Database**: PostgreSQL with Drizzle ORM
- **Features**: 
  - Bot configuration dashboard
  - Moderation command management (ban, kick, mute with permissions)
  - Music bot setup (YouTube, Spotify integration)
  - Anti-nuke protection settings
  - Server management tools
  - Real-time bot status monitoring
  - Keep-alive functionality
